# CLAUDE.md

## Сборка

При каждом запуске сборки Maven добавляй `clean`, то есть собирай через
`mvn clean package` / `mvn clean compile`, а не просто `mvn package` / `mvn compile`.

## Тесты

Никогда не запускай тесты без явной просьбы, не ищи их и не пиши. Ограничивайся
только компиляцией.

## Git

Никогда не делай `git push` и не спрашивай разрешения на push. После коммита
напоминай пользователю, что пушить он должен сам.

## Документация (ТЗ и README)

После очередного изменения кода, перед коммитом, спрашивай пользователя, не нужно
ли обновить `ТЗ.md` и `README.md` под внесённые изменения.
